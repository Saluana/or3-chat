# Nature Theme

The **Nature** theme provides a calm, organic look intended for experimenting with the override system.

Highlights:

- Soft glassmorphism surfaces with leafy gradients.
- Custom button variants (`forestSolid`, `forestOutline`, `mossGhost`) combined with identifier overrides for chat controls.
- Context-specific button styling for the sidebar and chat composer.
- State-based overrides for hover and disabled aesthetics.
- Tailored modal and input treatments that showcase the new theming pipeline.

Use `switchTheme('nature')` via the theme plugin or the UI to apply it at runtime.***
